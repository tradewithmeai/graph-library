About This Example

This example demonstrates live data streaming with SolVX Graph Engine:

  • RandomWalkSource generates live candle data
  • Updates every 250ms with 5-second candles
  • CrosshairTooltipPlugin shows OHLCV data
  • MovingAveragePlugin displays 20-period MA
  • Chart updates smoothly at 60 FPS

Try: Click "Stop" to pause updates, "Start" to resume
