About This Example

This example demonstrates the ShapesOverlayPlugin capabilities:

  • Trading session rectangles (Asian, London, New York)
  • Support and resistance zones as horizontal bands
  • Semi-transparent fills with colored borders
  • Multiple shape types on a single chart

Use cases: Mark trading sessions, key price levels, events, patterns
